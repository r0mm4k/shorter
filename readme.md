# Url-shorter API

> Backend API for Url-shorter

## Usage

Rename "config/.env.example" to "config/.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
